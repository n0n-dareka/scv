# SCV – Speculative Creation Visible

SCV は、テキストに潜んでいる「構造・感覚・出自」を三層で可視化するための小さなツールです。  
GitHub Pages 上のツールページから、テキストを入力して AI による読み解きを行えます。

## Demo

SCV ツールはこちらから利用できます：

https://n0n-dareka.github.io/scv/scv.html

## What SCV does

- テキストを入力すると、AI に渡すためのプロンプトを生成します
- AI が返した JSON を貼り付けると、構造・感覚・出自の三層で現します。
- Seed / Flow / Echo / Tide の 4 つの言葉は、三層を読み解く際の “補助的な視点” として扱えます
Seed｜どこから生まれたか（出自）
Flow｜動きを現した存在（構造）
Echo｜どこが響いたか（感覚）
Tide｜どこへ流すか（行方）
- 創作物の「見えない部分」を整理するための補助ツールとして使えます。
